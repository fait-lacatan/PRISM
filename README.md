# Discover PRISM: Installation Guide

This document provides instructions for installing and running the **Discover PRISM** application across different platforms.

## 💻 Windows (Recommended)
The most professional way to experience the app is via the Windows Installer.
1.  Locate **`Discover_PRISM_Setup.exe`**.
2.  Double-click to run the installer.
3.  **Note**: If Windows SmartScreen appears ("Windows protected your PC"), click **More info** and then click **Run anyway**.
4.  The installer will create a Desktop shortcut and a Start Menu entry under "Discover PRISM".

---

## 🍎 macOS
Since this is an academic submission and not a signed App Store release, Apple's security will attempt to block the app. 
1.  Unzip the **`Discover_PRISM_MacOS.zip`** file.
2.  Move the **`Discover PRISM.app`** bundle to your **Desktop** or **Applications** folder.
3.  **CRITICAL**: Do not double-click the app normally. Instead:
    - **Right-click** (or Control-click) the app icon.
    - Select **Open** from the menu.
    - A pop-up will appear saying the developer cannot be verified. Click **Open** again.
4.  The app will now launch. This step only needs to be performed once.

---

## 🤖 Android
1.  Transfer **`prism.apk`** to your Android device.
2.  Locate the file using a File Manager and tap to install.
3.  If prompted, allow "Install from Unknown Sources" in your device settings.
