# CF-PAD Setup

The CF-PAD model definition source code has been removed from this repository to comply with external licensing and attribution requirements.

To run the CF-PAD networks, you must download the following files from the original repository:

**[https://github.com/meilfang/CF-PAD/tree/main](https://github.com/meilfang/CF-PAD/tree/main)**

Download and place these files in this directory:
- `common.py`
- `mixstyle.py`
- `model.py`
