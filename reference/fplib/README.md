# fplib (GT-521Fx2 fingerprint sensor module)

The source code for `fplib` has been removed from this repository to comply with external licensing and attribution requirements.

To use the minutiae extraction features, you must clone or download the `fplib-GT521Fx2` implementation from the original repository:

**[https://github.com/Ribin-Baby/fplib-GT521Fx2/tree/main](https://github.com/Ribin-Baby/fplib-GT521Fx2/tree/main)**

Place the downloaded source files directly in this directory (`reference/fplib/`) so that the code can run.
