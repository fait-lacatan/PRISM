# Security Policy

## Security Architecture & Threat Model
For a detailed analysis of PRISM's defense-in-depth architecture, key hierarchy, and attack vector mitigations, please refer to our **[THREAT_MODEL.md](./THREAT_MODEL.md)**.

## Reporting a Vulnerability

We value the contributions of the security community. If you discover a flaw in PRISM, please contact the maintainers for private reporting. We request that you refrain from opening public issues to prevent potential exploitation while a fix is developed.
